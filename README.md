# quick-shell
Spawn a shell by creating a tunnel to the container running GitHub Actions workflow

## Usage
- Fork this repo
- Click on `Actions` tab
- Select `Spawn a shell` workflow
- Click on `Run workflow` button
- Choose Runner
- Click on `Run workflow` button
- Check logs for access url
